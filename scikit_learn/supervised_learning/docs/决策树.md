# 决策树
*决策树是一种基本的分类与回归方法。决策树模型呈现属性结构，在分类问题中，
表示基于特征堆实例进行分类的过程。它可以认为是if-then规则的集合，也可以
认为是定义在特征空间与类空间上的条件概率。*

*决策树学习通常包括3个步骤：特征选择、决策树的生成和决策树的修剪。*

*决策树的学习思想有：ID3算法、C4.5算法和CART算法*
